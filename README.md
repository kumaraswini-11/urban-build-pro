## ‚ú® Key Features

- **Industrial Design System:** Sharp corners (`radius: 0rem`), heavy borders, and structural layouts reflecting engineering precision.
- **OKLCH Color Space:** Advanced color management using `oklch()` for consistent perceived brightness across light and dark modes.
- **Performance Optimized:** Optimized images via `next/image`, font-swapping for Zero Layout Shift, and native smooth scrolling.
- **Interactive FAQ:** Smooth, staggered-entrance accordion built with `AnimatePresence` and Motion.
- **Mosaic Work Gallery:** A grid-based portfolio featuring hover-state slide-up animations and high-fidelity project images.
- **Modern Forms:** Accessible contact and newsletter forms prepared for React 19 Server Actions.

## üõ†Ô∏è Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone [https://github.com/kumaraswini-11/urban-build.git](https://github.com/kumaraswini-11/urban-build.git)
   cd urban-build
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Environment Variables:**
   Create a `.env` file based on `.env.example`:

   ```bash
   cp .env.example .env
   ```

4. **Run the development server:**

   ```bash
   npm run dev
   ```
